# 🛍️ ShopEZ – Full-Stack E-Commerce Platform

> **Tech Stack:** React.js · Node.js · Express.js · MongoDB · JWT Auth · Vite

---

## 🚀 Quick Start

### Prerequisites
- **Node.js** v18+
- **MongoDB** running locally on port 27017

### 1. Start the Backend
```bash
cd server
npm run dev
```

### 2. Seed the Database (first time only)
```bash
cd server
npm run seed
```

### 3. Start the Frontend
```bash
cd client
npm run dev
```
